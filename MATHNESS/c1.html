3)COMPUTATION OF NUMBERS IN STANDARD FORM

  • TWO NUMBERS IN STANDARD FORM CAN BE ADDED OR SUBTRACTED IF BOTH NUMBERS HAVE THE SAME INDEX a x 10ⁿ + b x 10ⁿ = (a + b) x 10ⁿ

3.5 x 10³ + 2.103 x 10³ = (3.5 + 2.103) x 10³ = 5.603 x 10³

5.89 x 10⁻³ – 2.01 x 10⁻³ = (5.89 – 2.01 ) x 10⁻³ = 3.88 x 10⁻³

5.6 x 10⁵ + 2.41 x 10⁷ = 0.056 x 10⁷ + 2.41 x 10⁷ = (0.056 + 2.41) x 10⁷ = 2.456 x 10⁷

3.4 x 10⁻⁴ – 8 x 10⁻⁵ = 3.4 x 10⁻⁴ – 0.8 x 10⁻⁴ = (3.4 – 0.8 ) x 10⁻⁴ = 2.6 x 10⁻⁴

  • WHEN TWO NUMBERS IN STANDARD FORM ARE MULTIPLIED OR DIVIDED, THE ORDINARY NUMBERS ARE MULTIPLIED OR DIVIDED WITH EACH OTHER WHILE THEIR INDICES ARE ADDED OR SUBTRACTED

(a x 10³) x (b x 10⁵) = (a x b) x 10³⁺⁵ = ab x 10⁸

(a x 10⁹) ÷ (b x 10⁵) = (a ÷ b) x 10⁹⁻⁵ = a/b x 10⁴

9.5 x 10³ x 2.2 x 10² = (9.5 x 2.2) x 10³⁺² = 20.9 x 10⁵ = 2.09 x 10¹ x 10⁵ = 2.09 x 10¹⁺⁵ = 2.09 x 10⁶

(7.2 x 10⁵) ÷ (6 x 10⁻²) = (7.2/6) x 10⁵⁻⁽⁻²⁾ = 1.2 x 10⁵⁺² = 1.2 x 10⁷
